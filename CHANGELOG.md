# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- Universal multi-platform downloader via yt-dlp
- Video & audio download support (MP4, MP3, FLAC, WAV, M4A)
- Playlist merge to single FLAC file
- Smart resolution detection
- Real-time download progress (SSE)
- Dynamic UI theme by platform
- High Compatibility Mode (H264/AAC re-encode)
- File manager (list & delete downloads)

### Notes
This version marks the first stable release of UniviDown.
